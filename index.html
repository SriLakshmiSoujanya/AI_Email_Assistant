AI-Powered Email Assistant

Smart email support system with Flask, Streamlit, OpenAI & SQLite

📌 About the Project

The AI Email Assistant helps automate support emails by:

  • 📧 Fetching emails via IMAP
  • 📝 Classifying them (Support / Query / Request / Help)
  • 🤖 Generating AI-powered replies (RAG with OpenAI)
  • 📊 Showing insights in a dashboard (Streamlit)

🚀 Features

  • Sentiment & priority classification
  • Automatic reply suggestions
  • SQLite storage for emails & responses
  • Interactive Streamlit dashboard
